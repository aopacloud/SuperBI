## 介绍

SuperBI 是一款交互简单、分析高效、操作灵活、查询快速的多维可视化分析平台。
核心功能点包括：

- 自助分析 - 业务人员自由拖拽数据自助分析
- 可视化报表 - 基于丰富的图表类型及组件快速搭建可视化报表
- 秒级查询 - 支持对海量数据多种数据源进行秒级查询
- 权限控制

## 快速上手

### 安装

> 前端项目依赖 NODE 环境，请确保本地 NODE 正确安装

```shell
# 拉去仓库
git clone https://github.com/olaola-chat/superbi.git

# 进入项目目录
cd frontend

# 安装依赖
npm install

# 运行
npm run dev

# 构建
npm run build

# 构建不同环境 运行不同biuld命令
npm run build:*
```
